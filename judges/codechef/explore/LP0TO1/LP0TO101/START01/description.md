# Problem Statement

Write a program that accepts a number, **n**, and outputs the same.

# Constraints

* **0 ≤ n ≤ 10<sup>5</sup>**

# Format

## Input

The only line contains a single integer. 

## Output

Output the answer in a single line.

# Sample Testcases

## Sample testcase 1

**Sample Input**

```
123
```

**Sample Output**

```
123
```
